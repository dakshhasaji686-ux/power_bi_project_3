## Overview
This Power BI dashboard provides a clear and simple view of sales performance, including profit, cost, and trends across categories, regions, and time. It is designed to help understand business performance and support better decision-making.

## Key Metrics
- Total Sales: 848K
- Total Cost: 686K
- Total Profit: 162K
- Return Rate: 0.10
- Average Sales: 848.40
- Last 3 Month Sales: 210K
- Monthly Sales Growth: 848K

## Features

### Profit Analysis
- Quarterly profit comparison
- Region-wise profit distribution

### Category Analysis
- Sales distribution by category (Electronics, Books, Clothing, etc.)
- Cost vs Profit comparison for each category

### Monthly Trends
- Monthly sales and quantity trends
- Helps identify patterns and seasonality

### Filters
- Category filter
- Region filter
- Year (YTD) filter

### Detailed View
- Displays selected category details including:
  - Sales
  - Profit
  - Quantity

## Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
